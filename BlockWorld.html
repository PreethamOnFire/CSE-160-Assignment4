Notes for Graders
I just finished watching the show Severance and I thought it be fun to try and design my blocky world to look like the severed floor on Severance. I tried making the office to be basic and maze like to reflect how the office was in the show.

Features:
You can control the camera using the mouse and use WASD to move. E and Q can also rotate the camera. You can place blocks using middle mouse button and break blocks using left click. You can only break and place wall blocks.
I created a simple object loader that assumes that the faces are all triangles. I modeled a simple desk in blender and exported it as a obj file. I loaded it into the world and you can find that desk model around the world. You can find the obj file in the repo.
Also I do have colors and textures working together. The sky box is a blue box and there is hallway that is a black hallway.

For Lighting:
You can change the color of the light with the RGB sliders. You can change the location of the point light using the XYZ sliders. In order to use the sliders you need to disable the animation. All the buttons are toggles. Normal Vizualizations sets the color to the normal, Animation stops and starts the animation of the point light, and lights turns on and off the light. The mouse control button disables using the mouse to rotate the camera to make it easier to select the options. The spotlight can be changed to a flashlight where the spotlight location becomes the camera location and the spotlight points to where ever the camera is pointing.

Please use browser that supports "canvas"
